
## Distribution?

The possible values a variable can take
and how frequently they occur

# types?
### 1-Discrete Distributions
 -countable values
 -outcomes are separate numbers
    :Rolling a die

## -2 Continuous Distributions
-when the variable can take an infinite number of possible values
-measured rather than counted
     :Time


## Conditional Probability
-P(A∣B)
the probability of A happening if B already happened


## Multiplication Rule
to find the probability that 2 events happen together


## Bayes' Rule?
 used to update probabilities when we get new info
 - to find the probability of a cause after observing the  result 

## Independence:
2 events independent if one event doesn't affect the other one
P(A∩B)=P(A)×P(B)


## Mutually Exclusive
events that can't happen at the same time
: getting 2 numbers when rolling 1 die


#### Uniform Distribution
 all outcomes have the same probability 
  : rolling die

#### Bernoulli Distribution
 the simplest distribution where there is one experiment with only 2 possible outcomes
  :flip a coin 


#### Binomial Distribution
sequence of Bernoulli Distribution
but there is conditions:
 - n= fixed numbers of trial
 - p= same probability of success 
 - the trials are independent 
 : flip a coin 10 times 


#### Poisson Distribution
λ=mean=variance
how many times the event happens in fixes time
