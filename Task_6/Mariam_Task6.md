
## Histogram

  For numerical data
  bars touch each other
  each bar represents the number of values inside that interval

  ![[laerd-statistics-example-histogram-frequencies-for-age 4.png]]

## Bar Chart

  used for categorical data each bar represents a category and the height of the bar shows its frequency
  The bars are separated 
  mainly used to compare different groups

![[barchart_ver_1.jpg|305]]


## Dot Plot

  Each dot = one observation
  not practical for large datasets
  ![[dotHist1-768x669.png|279]]


## Box and Whisker Plot

  summarizes the data using the five-number summary 
   minimum,  maximum, Q1, median(Q2), Q3, 
  The whiskers extend to the smallest and largest values
![[whm_elements_of_a_boxplot_en_wikimedia.png|528]]



## Pie Chart

each slice shows how much one category contributes to that total
It is most effective when there are only a few categories because it allows to quickly compare proportions visually
![[images.jpg|298]]


## violin plot

 A violin plot is a box plot + density "distribution shape" of the data
 gives more detailed information about how the data is distributed
 ![[zcKyA.png|605]]



# Symmetry and Skewness

A distribution is symmetric when both sides look the same
A distribution is skewed when one side is longer than the other
## In a symmetric distribution:

The mean and median are equal.
There is no long tail on either side

![[7588d6507c1315bd04332b326df7bec5.png|371]]

## skewed Distribution:
2 types:

1-Right-Skewed 

 The tail extends to the right.
 Mean > Median
 
![[Right-skewed-distribution.jpg|364]]


2-Left-Skewed
 The tail extends to the left.
 Mean < Median
 
![[Left-skewed-distribution.webp|368]]
