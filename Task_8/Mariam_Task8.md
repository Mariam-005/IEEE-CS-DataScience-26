
## Random variable?
number that represents the result of a random experiment 
Why?
so we can work with the outcome mathematically
Types ?
##### 1.Discrete
values that can be counted ,separated
0, 1, 2, 3, 4....
##### 2 Continuous
 can take any value within a range
  there are many possible values


##  Probability Distribution
shows:
- all the possible values 
- the probability of each value

#### Discrete Probability distribution
 can list the probabilities in a table or chart
(usually histogram)

#### Continuous Probability distribution
 use something called a "Probability Density Function"
 instead of bars get a curve

calculate probability for a range of values
probability equals the area under the curve


## Binomial distribution?
repeated experiment when there are only two possible outcomes of the experiment 
 : flip a coin 10 times 

conditions:
- n= fixed numbers of trial
 - p= same probability of success 
 - the trials are independent 


##### Shape of the Binomial Distribution?
  depends on the probability of success
 If p = 0.5 
    the distribution is symmetrical
if p is bigger or smaller than 0.5
     the distribution becomes skewed


### Expected value E(X)
average outcome if we repeat the experiment many times
how to calculate?
the sum of results of multiplying each value by its probability

for binomial distributions the formula is 
E(X) = n × p

### Law of large numbers:
when we repeat an experiment many times the average result gets closer to the expected value

If we flip a coin:
10 times : results may look random
100 times : the number of heads will be closer to 50%
1000 times : it will be more closer to 50%


#### Gambler's fallacy?
when someone believes that past results affect future results in random events

(when the trial is independent the probability stays the same)
