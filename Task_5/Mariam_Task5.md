
# Qualitative and Quantitative Data

· Qualitative data: descriptions without numbers
· Quantitative data: numbers obtained by
   counting or measuring
· Discrete/Continuous data are types of quantitative data
· Discrete data: Only certain values 
 **Example**  
  Number of students 
· Continuous data: Almost any value
 **Example**  
   Height (170.5 cm)

# Levels of Measurement
## 1- Nominal Data 
most basic type of data
can be categorized 
but
can not be ranked
 **Example:**  
  gender(male/female)
## 2-Ordinal data
can be categorized  and ranked

**Example:**  
  Levels of education

## 3-interval data
 Ordered
 Equal intervals
 No absolute zero
**Example:** 
Temperature in Celsius (20°C, 30°C)
## 4- Ratio data
Ordered  
Equal intervals  
Absolute (true) zero
**Example:** 
Weight in kg 
   
# What is a hypothesis?
 hypothesis is an idea that can be tested
- Null =No difference
- Alternative =There is a difference 

- If the difference is statistically significant → Reject H₀.
- If not significant → Fail to reject H₀.
## **P-value:**
The probability of obtaining the observed sample results (or more extreme) assuming the **null hypothesis is true**
- Small p-value (≤ 0.05) → Strong evidence against H₀ → Reject H₀
- Large p-value (> 0.05) → Weak evidence against H₀ → Fail to reject H₀


**Two-sided test:** considers extreme results in both directions.  
**One-sided test:** considers one direction only and must be used carefully.

## **Confidence Interval:**  
A range of values, calculated from a sample, that is likely to contain the true population parameter


- **Bigger sample → smaller interval**  
    
- **More variability → bigger interval**  
    
- **Higher confidence → bigger interval**


## **Regression:**
A way to understand the relationship between variables and make predictions.

**Dependent variable:** the outcome we want to predict.  
**Independent variables:** the factors we use to make the prediction.

**Simple linear regression:** when there’s only one predictor.  
**Multiple linear regression:** when there are multiple predictors.  
**Logistic regression:** when the outcome is a choice between two options (Yes/No).
